# NideAdmin

基于 Vue.js + iView 的后台管理基础框架。

+ [线上体验](http://nideadmin-demo.nideshop.com/)
+ [开发文档](http://nideadmin.nideshop.com/)

### 使用方法
```
# 进入客户端目录
cd vue-client

# 安装依赖
npm install --registry=https://registry.npm.taobao.org

# 启动开发服务
npm run serve

# 构建生产环境
npm run build
```
